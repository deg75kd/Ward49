Data Science Portfolio

My name is Kevin DeJesus, and I aspire to be a data scientist. For the past 9 years I have worked as a database administrator. In the summer of 2018 I started researching other data-related opportunities and was immediately drawn to data science. I have now completed 2 online courses with Coursera and Udacity. Those courses and independent study have helped me build the skills and knowledge to complete the following projects. As I continue to grow as a data scientist, I plan to add to my portfolio.

Song Lyrics Genre Predictor

This project presents my efforts to categorize song lyrics using data science. First, I cleaned my data source and uploaded the results to a PostgreSQL database. I performed some analysis on that data and created a few graphs. Next I built a machine learning pipeline using natural language processing and a random forest classifier. Then I connected to the ChartLyrics API to find lyrics of other songs. Finally, I built a web site around it all to allow users to apply the model to their favorite songs.

IBM Recommendations

In this Udacity assignment I was given real data from the IBM Watson Studio platform. The end goal was to develop a recommendation system for users based on the articles they have read. First I performed exploratory data analysis. Then I created a way to provide rank-based recommendations. Next I utilized user-user based collaborative filtering. Finally, I tested matrix factorization and provided a conclusion.

Chicago Restaurant Recommender

In this capstone project the only requirement was to use Foursquare's API to retrieve data. I decided to build a recommendation engine for restaurants in Chicago. I used a dataset from the city's website to determine neighborhood names and locations. Then I used the API to build a list of restaurants and checked each venue for users who have visited. Finally I built the recommender and mapped the results for a couple of users.

WorldBank Dashboard

For this Udacity assignment I performed analysis on data from the World Bank and created several graphs detailing my findings. Then I used to Flask, Plotly and Bootstap to deploy it as a web page hosted on Heroku.

Opportunities for Airbnb Hosts in Chicago

In this Medium blog post I explain my efforts to analyze Airbnb data from the city of Chicago. First I explored the data to answer a couple of questions for property owners looking to become hosts. Then I applied linear regression to the dataset to predict prices. Unfortunately, the results were very poor. This project would be a good candidate to revisit in the future.

Kaggle Titanic Project

In this notebook I used machine learning techniques to predict the survival of passengers on the Titanic. First I explored the data and prepared it for use by the machine learning models. Then I tested several classifiers and used cross validation to determine the model with the best accuracy. Finally, I appled the best performing model to the test dataset to get predictions.
